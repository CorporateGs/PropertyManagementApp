## üö¶ Getting Started

### Installation
```bash
cd property_management_system/app
npm install
```

### Environment Variables
```env
DATABASE_URL="postgresql://user:pass@localhost:5432/propmanage"
ANTHROPIC_API_KEY="your-key-here"
STRIPE_API_KEY="your-key-here"
TWILIO_ACCOUNT_SID="your-sid-here"
TWILIO_AUTH_TOKEN="your-token-here"
SENDGRID_API_KEY="your-key-here"
```

### Database Setup
```bash
npx prisma generate
npx prisma db push
npx prisma db seed
```

### Run Development Server
```bash
npm run dev
```
